## 本地开发启动（简版）

- 后端：进入 `pure-admin-backend` 执行 `pnpm install`、`pnpm start`

- 父前端：进入 `pure_admin_antv_g6_phar_2025_04_07` 执行 `pnpm install`、`pnpm dev`（端口 8848）
